# Supervised Learning – Decision Tree Classifier on Iris Dataset

This is a **beginner-friendly machine learning project** using Python's `scikit-learn`.  
We use the **Decision Tree Classifier** to predict the type of Iris flower based on its features.

---

## **Dataset**
We are using the **Iris dataset** from `sklearn.datasets`, which contains:
- 150 samples
- 4 features: sepal length, sepal width, petal length, petal width
- 3 classes: Setosa, Versicolor, Virginica
